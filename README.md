# A-Tower_Defense

一个使用Bevy引擎开发的简单塔防游戏。

## 游戏功能

- **地图系统**: 10x8的网格地图，包含路径和可建造区域
- **敌人系统**: 敌人沿着预设路径移动，有生命值和速度属性
- **塔防系统**: 可建造防御塔，自动攻击范围内的敌人
- **子弹系统**: 塔防发射子弹攻击敌人
- **游戏状态**: 金钱、生命值、波数管理
- **UI显示**: 控制台显示游戏状态信息

## 游戏玩法

1. **建造塔防**: 点击可建造区域（绿色格子）建造防御塔（消耗50金钱）
2. **防御敌人**: 敌人会沿着灰色路径移动，到达终点会减少生命值
3. **升级系统**: 当前版本支持基础塔防建造
4. **波数系统**: 每波敌人数量逐渐增加

## 控制方式

- **鼠标左键**: 在可建造区域点击建造防御塔

## 运行游戏

```bash
cd A-Tower_Defense
cargo run
```

## 游戏界面

- 灰色格子: 敌人路径
- 绿色格子: 可建造区域
- 蓝色圆形: 敌人
- 黄色圆形: 防御塔
- 红色圆形: 子弹

## 开发说明

- 使用Rust语言和Bevy游戏引擎
- 基于ECS（实体组件系统）架构
- 模块化设计，易于扩展
